Weather App

The weather app is a simple, user-friendly web application built with HTML, CSS, and JavaScript. It provides real-time weather information for cities worldwide, either through manual search or geolocation. The app displays key weather details like temperature, humidity, wind speed, and conditions using a clean, intuitive interface. JavaScript powers data retrieval from a weather API and updates the UI dynamically. Features may include interactive weather icons, temperature graphs, and multi-day forecasts. Users can switch between Celsius and Fahrenheit and explore more weather details.
